4.2 Devise a strategy for filling in all of the missing values in the dataset. The strategy does not need to be sophisticated. For example, you could use the mean/median for that day, or the mean for that 5-minute interval, etc.

Strategy choosen: Replace the NAs with the mean value for the corresponding 5-min interval. This values were obtained previously and stored in the variable avg_by_interval, from which they will be extracted.

m<-dim(activity)[1]
activity_clean<-activity
for (i in 1:m){
        if(is.na(activity_clean[i,1])){
                searcher<- activity_clean[i,3]
                result<-avg_by_interval[avg_by_interval$interval==as.integer(searcher),2]
                activity_clean[i,1]<- result
        }
}
